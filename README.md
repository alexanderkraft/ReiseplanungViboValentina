# Reiseplaner - Frankfurt → Sant'Agata di Esaro

Interaktive Reiseplanung für Autofahrten in Europa, mit Tanken, Hotels, Restaurants

## Features

- Interaktive Karte mit Leaflet.js (4 Marker + 3 Routen)
- Live Budget-Rechner (Maut, Benzin, Hotels)
- Etappen-Cards mit Zoom-Funktion
- Dark/Light Mode Toggle
- CSV Export
- Mobile Responsive

## Route
Vorschlag verschiedener Routen
Etappensizing

## Budget (geschätzt)
Budgetanalyse BEnzin/Diesel, Hotels, Restaurants, Maut, etc auf der Strecke

## Tech Stack

- HTML5 / CSS3 (Glassmorphism Design)
- Vanilla JavaScript
- Leaflet.js (OpenStreetMap)
- Font Awesome Icons
